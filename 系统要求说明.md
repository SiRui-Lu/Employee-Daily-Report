# 校园洗涤员工日报系统 - 安装说明

## 📦 安装步骤

1. **双击运行** `校园洗涤员工日报系统 Setup 1.0.0.exe`
2. **选择安装位置**（可选，默认安装在 `C:\Users\用户名\AppData\Local\校园洗涤员工日报系统`）
3. **点击"安装"按钮**
4. **等待安装完成**
5. **安装完成后**，可以从以下位置启动应用：
   - 桌面快捷方式
   - 开始菜单 → 校园洗涤员工日报系统

## 💻 系统要求

### 最低要求
- **操作系统**：Windows 7 或更高版本（推荐 Windows 10/11）
- **系统架构**：64位（x64）Windows 系统
- **内存**：至少 2GB RAM（推荐 4GB 或更多）
- **磁盘空间**：至少 200MB 可用空间
- **显示器分辨率**：至少 800x600（推荐 1024x768 或更高）

### 重要说明
- ✅ **不需要安装 Node.js、Python 或其他开发工具**
- ✅ **不需要安装浏览器插件**
- ✅ **不需要网络连接**（应用完全离线运行）
- ✅ **不需要管理员权限**（普通用户即可安装和使用）

### 不支持的系统
- ❌ 32位（x86）Windows 系统
- ❌ Windows XP 及更早版本
- ❌ Linux 或 macOS（当前版本仅支持 Windows）

## ⚠️ 常见问题

### Q: 安装时提示"此应用无法在你的电脑上运行"？
**A:** 这通常是因为：
- 您的系统是 32 位，而安装包是 64 位版本
- 请检查您的系统：右键"此电脑" → 属性 → 查看"系统类型"

### Q: 安装时被杀毒软件拦截？
**A:** 这是正常现象，因为安装程序需要写入系统文件。请：
- 选择"允许"或"信任此程序"
- 或将安装程序添加到杀毒软件白名单

### Q: 安装后无法启动？
**A:** 请尝试：
- 右键安装程序 → 以管理员身份运行
- 检查是否被杀毒软件阻止
- 重新安装应用

### Q: 可以卸载吗？
**A:** 可以，通过以下方式卸载：
- 控制面板 → 程序和功能 → 找到"校园洗涤员工日报系统" → 卸载
- 或：开始菜单 → 找到应用 → 右键 → 卸载
